Política de Privacidad


PyDownTV no almacena ninguna información sobre el usuario que se conecta a la web.

Además, PyDownTV no almecana ninguna Cookie en tu navegador.

PyDownTV SÍ dispone de una Base de Datos para mantener un registro histórico de los vídeos que se descargan a través de la web, pero en ningún caso se guarda información relacionada sobre el usuario que descargó el vídeo. En esta base de datos se guardan:

Campos BBDD
Fecha (día, hora, ...)
URL de la miniatura del vídeo
URL introducida por el usuario
Título del vídeo

PyDownTV puede acceder a varios datos del usuario del servicio proporcionados por su IP y su navegador, estos datos solo son utilizados para proporcionar una mejor experiencia al usuario y en ningún caso se guardan en ningún registro. Solo son utilizados en cada conexión particular. Los datos a los que actualmente tiene acceso PyDownTV se pueden consultar en el siguiente apartado: Consulta los datos que PyDownTV conoce sobre ti.

PyDownTV es Software Libre y se puede comprobar su código aquí.
